- MVC
  - 软体架构风格
    - 不是标准
  - 将数据操作，业务处理，界面展示进行了拆分
  - 核心思想： 解耦合
    - 松耦合
  - Model
    - 模型
    - 封装数据的交互操作
      - CRUD（Create Retreive Update Delete）
  - VIew
    - 视图
    - 用来将数据呈现给用户
  - Controller
    - 控制器
    - 接受用户的输入
    - 用来协调model和view的关系，并对数据进行操作，筛选
  - 流程
    - 控制器接受用户请求
    - 调用模型，获取数据
    - 控制器将数据展示到视图中
