#### the fuck

- 一个终端指令修复工具
- 当指令在输入错误的时候，我们可以通过fuck进行弥补
  - 可以提供指令修复方案
  - enter
  - contro+c
  - ⬆️⬇️
- 使用
  - 安装前置依赖python3-dev
  - Python3-pip
  - 安装thefuck插件
    - Pip3  install thefuck
  - 修改别名
    - 在环境变量汇中修改别名
